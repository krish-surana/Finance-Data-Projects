Relative Valuation | Indian Defense Sector

Self initiated project comparing four listed Indian defense PSUs on relative valuation, using public data sources instead of a paid terminal.

Companies covered


Hindustan Aeronautics Ltd (HAL)
Bharat Electronics Ltd (BEL)
Mazagon Dock Shipbuilders Ltd (MAZDOCK)
Bharat Dynamics Ltd (BDL)


What this project does


Builds a peer comparison table on EV/EBITDA, P/E, P/BV, and ROE
Flags Bharat Dynamics trading at a steep valuation premium to peers despite a weaker return profile
Uses live Excel formulas so the model recalculates if the raw data is refreshed, instead of hardcoded numbers


Files


Defense_Sector_Relative_Valuation.xlsx - the model, three sheets:

Raw_Data: sourced financial data with citations for every number
Comps_Analysis: comp table with formulas pulling from Raw_Data, peer average, and premium/discount calculation
Summary_Insights: key findings, methodology, and limitations





Data sources

All data is from public sources, not Bloomberg Terminal or CapIQ:


MarketsMojo (Q4 FY26 result analysis, BDL)
Screener.in (Mazagon Dock company page)
Yahoo Finance (peer market data)
BusinessToday (brokerage estimates for BEL)


Full citations with dates and links are in the Raw_Data sheet.

Key finding

Bharat Dynamics trades at an EV/EBITDA of roughly 177x, about five times the peer average set by HAL and Mazagon Dock (~34x), despite having the lowest ROE in the peer set and a year on year decline in quarterly profit. This points to the market pricing in a strong execution recovery that has not yet shown up in results.

Limitations


Data points were pulled on different dates (April 2025 to May 2026) because free public sources do not all update at the same time. This is a same period comparison, not a same day snapshot.
A few metrics were not available from public sources for all four companies and were left blank rather than estimated.
